# Customer Segmentation Using K-Means Clustering

## Project Overview

This project is a Machine Learning based Customer Segmentation System developed using Python, Flask, and K-Means Clustering.

The system groups customers into different segments based on:
- Income
- Spending behavior
- Age
- Children
- Recency

The goal of this project is to help businesses:
- Identify valuable customers
- Improve marketing strategies
- Increase customer satisfaction
- Make data-driven business decisions

---

# Features

- Customer Segmentation using K-Means Clustering
- Data Cleaning and Preprocessing
- Feature Engineering
- Elbow Method for Optimal Clusters
- Interactive Flask Web Application
- Professional Dashboard UI
- Customer Insights and Recommendations
- Cluster Visualization

# Installation Steps

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Customer-Segmentation-Project.git
```

---

## 2. Open Project Folder

```bash
cd Customer-Segmentation-Project
```

---

## 3. Create Virtual Environment

```bash
python -m venv venv
```

---

## 4. Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

---

## 5. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6. Run Flask Application

```bash
python app.py
```
